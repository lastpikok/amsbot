# AMSbot - Каталог автомобилей для Telegram Mini App

## 📋 Структура проекта

```
AMSbot/
├── server.js              # Сервер на Express
├── package.json           # Зависимости проекта
├── create-cars-xlsx.js    # Скрипт для создания тестового XLSX файла
├── cars.xlsx              # Файл с автомобилями (создаётся вручную или скриптом)
└── public/
    ├── index.html         # Главная страница
    ├── style.css          # Стили
    └── app.js             # Клиентский JavaScript
```

## 🚀 Запуск проекта

### 1. Установите зависимости
```bash
npm install
```

### 2. Создайте файл с автомобилями (опционально)
Для создания тестового файла с автомобилями выполните:
```bash
node create-cars-xlsx.js
```

Или создайте файл `cars.xlsx` вручную со следующими колонками:
- **Марка** (brand)
- **Модель** (model)
- **Год** (year)
- **Цена** (price)
- **Пробег** (mileage)
- **Двигатель** (engine)
- **КПП** (transmission)
- **Изображение** (image) - URL картинки
- **Описание** (description)

### 3. Запустите сервер
```bash
npm start
```

Сервер запустится на: **http://localhost:3000**

## 📝 Как добавить/удалить автомобиль

### Добавление:
1. Откройте файл `cars.xlsx` в Excel или аналогичной программе
2. Добавьте новую строку с данными автомобиля
3. Сохраните файл
4. Обновите страницу в браузере - автомобиль появится автоматически

### Удаление:
1. Откройте файл `cars.xlsx`
2. Удалите строку с автомобилем
3. Сохраните файл
4. Обновите страницу в браузере - автомобиль исчезнет

## 🌐 Telegram Mini App

Для использования в Telegram:
1. Загрузите сайт на хостинг (или используйте ngrok для локального сервера)
2. В Telegram создайте бота через @BotFather
3. Привяжите URL к Web App через @BotFather

## 🛠 API Endpoints

- `GET /api/cars` - Получить все автомобили
- `GET /api/cars/search?q=...` - Поиск автомобилей
- `GET /api/brands` - Получить список марок
- `GET /api/cars/brand/:brand` - Фильтр по марке
