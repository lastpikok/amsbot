# 🚀 Быстрая загрузка фото на GitHub

## Шаг 1: Создайте репозиторий

1. https://github.com/new
2. Название: `amsbot-cars`
3. **Public** ✅
4. Create repository

## Шаг 2: Подготовьте фото

Для каждого автомобиля создайте папку с названием вида:
**`Марка Модель (последние_6_цифр_VIN)`**

Примеры:
```
📁 Hyundai Elantra (131596)/
   ├── 1.jpg
   ├── 2.jpg
   └── 3.jpg

📁 Audi Q3 (720830)/
   ├── 1.jpg
   └── 2.jpg

📁 HAVAL CHITU (020320)/
   └── 1.jpg
```

**Как узнать VIN:** Откройте файл `cars.xlsx` и найдите нужный автомобиль в колонке VIN.

## Шаг 3: Загрузите фото

### Самый простой способ (через сайт):

1. Откройте https://github.com/ВАШ_USERNAME/amsbot-cars
2. Нажмите **Add file** → **Create new file**
3. В поле имени введите: `photos/Hyundai Elantra (131596)/1.jpg`
4. Перетащите фото или нажмите "choose your files"
5. Нажмите **Commit changes**
6. Повторите для остальных фото

### Для большого количества (через GitHub Desktop):

1. Установите: https://desktop.github.com
2. Clone репозиторий
3. Скопируйте папки с фото в репозиторий
4. Сделайте Commit → Push

## Шаг 4: Проверьте

1. Откройте сайт: http://localhost:3000
2. Кликните на автомобиль
3. Фото должны отобразиться в модальном окне

---

## 📝 Важно

- **Название папки**: `Марка Модель (VIN)` например `Hyundai Elantra (131596)`
- **Названия файлов**: `1.jpg`, `2.jpg`, `3.jpg`...
- **Форматы**: `.jpg`, `.png`, `.webp` для фото; `.mp4` для видео
- **Размер файла**: до 25 MB

## 🎬 Видео

Для видео создайте папку `videos/` с такой же структурой:

```
videos/
├── Hyundai Elantra (131596)/
│   └── 1.mp4
└── Audi Q3 (720830)/
    └── 1.mp4
```
