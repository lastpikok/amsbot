## ✅ НЕ НУЖНО переименовывать файлы!

Сайт автоматически загрузит **все фото** из папки `Photo/` с любыми именами.

## Как загрузить фото

### Способ 1: Через сайт GitHub (рекомендуется)

1. Откройте https://github.com/lastpikok/amsbot-cars
2. Нажмите **Add file** → **Create new file**
3. Введите полное имя файла с путём:
   ```
   AUDI A3L (663982)/Photo/微信图片_20251221133513.jpg
   ```
4. Перетащите фото или выберите файл
5. Нажмите **Commit changes**
6. Повторите для остальных фото

### Способ 2: GitHub Desktop (для большого количества)

1. Установите: https://desktop.github.com
2. Clone репозиторий:
   ```bash
   git clone https://github.com/lastpikok/amsbot-cars.git
   cd amsbot-cars
   ```
3. Создайте структуру папок локально:
   ```
   amsbot-cars/
   └── AUDI A3L (663982)/
       └── Photo/
           ├── 微信图片_20251221133513.jpg
           ├── 微信图片_20251221133514.jpg
           └── ...
   ```
4. Сделайте Commit → Push

### Способ 3: Git CLI

```bash
git clone https://github.com/lastpikok/amsbot-cars.git
cd amsbot-cars

# Создайте папку
mkdir -p "AUDI A3L (663982)/Photo"

# Скопируйте все фото в папку
cp /путь/к/исходным/фото/*.jpg "AUDI A3L (663982)/Photo/"

# Отправьте на GitHub
git add .
git commit -m "Add photos for AUDI A3L (663982)"
git push
```

## Важные моменты

1. **Название папки автомобиля**: `БРЕНД МОДЕЛЬ (последние 6 цифр VIN)`
   - Пример: `AUDI A3L (663982)`
   - Пример: `HAVAL CHITU (020320)`

2. **Подпапка**: `Photo/` (с заглавной буквы, но код найдёт и с маленькой)

3. **Имена файлов**: **любые** — код загрузит все `.jpg`, `.png`, `.webp` файлы

4. **Ветка**: `master`

5. **Расположение**: Папки автомобилей **в корне репозитория** (не в `photos/`)

---

## Как узнать VIN

Откройте файл `cars.xlsx` → колонка **VIN** → последние 6 цифр

Примеры:
- VIN: `LFV2B2F34M7720830` → папка: `AUDI Q3 (720830)`
- VIN: `LGWEE4A59NL020320` → папка: `HAVAL CHITU (020320)`
