# Инструкция по загрузке фото и видео на GitHub

## 1. Создайте репозиторий

1. Зайдите на https://github.com
2. Нажмите **+** → **New repository**
3. Название: `amsbot-cars`
4. **Public** ✅
5. Создайте репозиторий

## 2. Структура папок

Создайте такую структуру в репозитории:

```
amsbot-cars/
├── photos/
│   ├── 720830/        (последние 6 цифр VIN)
│   │   ├── 1.jpg
│   │   ├── 2.jpg
│   │   └── 3.jpg
│   ├── 020320/
│   │   └── 1.jpg
│   └── ...
└── videos/
    ├── 720830/
    │   └── 1.mp4
    └── 020320/
        └── 1.mp4
```

## 3. Как узнать VIN

VIN указан в файле `cars.xlsx` в колонке VIN.

Используйте **последние 6 цифр** VIN для названия папки.

Пример:
- VIN: `LFV2B2F34M7720830` → папка: `720830`
- VIN: `LGWEE4A59NL020320` → папка: `020320`

## 4. Загрузка фото

### Вариант A: Через сайт GitHub (для небольшого количества)

1. Откройте репозиторий
2. Перейдите в папку `photos`
3. Нажмите **Add file** → **Upload files**
4. Создайте папку с именем VIN (например, `720830`)
5. Перетащите фото (назовите их `1.jpg`, `2.jpg`, `3.jpg`...)
6. Нажмите **Commit changes**

### Вариант B: Через GitHub Desktop

1. Установите GitHub Desktop: https://desktop.github.com
2. Clone репозиторий `amsbot-cars`
3. Создайте папки и добавьте фото локально
4. Сделайте Commit и Push

### Вариант C: Через git clone

```bash
git clone https://github.com/ВАШ_НИК/amsbot-cars.git
cd amsbot-cars

# Создайте папки
mkdir -p photos/720830
mkdir -p videos/720830

# Скопируйте фото
cp /путь/к/фото/*.jpg photos/720830/
cp /путь/к/видео/*.mp4 videos/720830/

# Отправьте на GitHub
git add .
git commit -m "Add photos for 720830"
git push
```

## 5. Настройка сайта

Откройте `public/app.js` и измените:

```javascript
const GITHUB_CONFIG = {
    username: 'pikok_o',    // ← Ваш GitHub username
    repo: 'amsbot-cars',    // ← Название репозитория
    branch: 'main'          // ← main или master
};
```

## 6. Проверка

1. Откройте сайт
2. Кликните на автомобиль
3. Если фото загружены — они отобразятся в модальном окне

## 7. Форматы файлов

- **Фото**: `.jpg`, `.jpeg`, `.png`, `.webp`
- **Видео**: `.mp4`, `.mov`, `.webm`

## 8. Ограничения GitHub

- Максимальный размер файла: **25 MB**
- Бесплатное хранилище: **1 GB** (на репозиторий)
- Для большого количества фото рассмотрите GitHub LFS
